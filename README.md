# Text-to-Image-App

A simple HTML, CSS and JS web app which uses [OpenAI's API](https://openai.com/) to convert text into images.

Requires an OpenAI account to generate an API key

## Front-end setup
`npm install`
`npm run dev`

## Back-end setup
`node server.js`
